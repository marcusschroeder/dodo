Welcome, scientist!

Almost all organisms acquired systems to scan their environment for sources of food or pheromones. This is crucial for the survival of any of them. For example, higher animals have developed complex olfactory systems to detect volatile components in the air (odorants). While the morphology of these systems has been revealed, is not yet fully understood how it encodes olfactory information.

Several in-vivo experiments explored the tuning profiles of olfactory receptors and the odorants that are activating them. These experiments are costly and error prone. To locate and identify receptors, their responses have to be compared to reference measurements from previous studies. This also means that a reference set of odorants has to be measured in each experiment without gaining new information. But, the major problem is that the number of measurable odorants per experiment is physiologically limited. Thus, it is desirable to reduce the size of this reference set. This is, where DODO comes in.

DODO discovers optimal odorant sets

DODO is designed to discover the the smallest possible set of odorants in a data-set by using elaborated optimization methods. This data-set can be either your data from previous experiments or data-sets provided by DODO.

These sets can then be used to identify and discriminate olfactory receptors based on their response spectra.
